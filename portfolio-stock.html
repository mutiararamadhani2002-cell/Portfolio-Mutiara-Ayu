Project Overview

This project focuses on forecasting stock price movement using statistical and machine learning time series approaches to support financial trend analysis and investment decision-making.

SARIMA and Prophet models were implemented to capture historical stock patterns, seasonality, and future forecasting trends, while interactive dashboards were developed for visualization and monitoring purposes.
